# 📖 Quiz Bíblico em Java

***Um projeto simples e interativo desenvolvido em Java, que simula um quiz de perguntas bíblicas diretamente no terminal.***

## 🚀 Sobre o Projeto

***O Quiz Bíblico é uma aplicação em linha de comando que testa o conhecimento do usuário sobre temas bíblicos através de perguntas de múltipla escolha.***

###Este projeto foi desenvolvido com foco em:

 - Praticar lógica de programação
 - Trabalhar entrada de dados com o usuário
 - Utilizar estruturas condicionais (if/else)
 - Melhorar organização e legibilidade de código

## 🛠️ Tecnologias Utilizadas
- ☕ Java
- 📥 Scanner (java.util.Scanner)
- 💻 Terminal / Console
- 🎮 Como Funciona

**O programa apresenta 5 perguntas bíblicas, cada uma com 4 alternativas.**

***O usuário deve:***

- Ler a pergunta
- Escolher uma opção (digitando o número correspondente)
- Receber o resultado ao final

***✔️ O sistema informa quais respostas estão corretas ou incorretas.***

## 📌 Exemplo de Execução
#### ===========|| QUIZ BÍBLICO ||============

#### RESPONDA AS PERGUNTAS BÍBLICAS ABAIXO:

#### PERGUNTA 01: Qual o nome do irmão de Moisés que foi chamado por Deus?

 1) Abnadabe
 2) Arão
 3) Davi
 4) Absalão

#### Sua Resposta: 2

## 📊 Regras do Jogo

- Cada pergunta possui apenas uma resposta correta
- O sistema valida suas respostas ao final
- Algumas perguntas exibem pontuação ao acertar
